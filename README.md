## Features

- Generate secure passwords  
- Choose length of password  
- Option to include:  
  - Uppercase letters  
  - Lowercase letters  
  - Numbers  
  - Symbols  
- Copy generated password to clipboard

## Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/GAYATHRI1006/Password_Generator_App.git
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Run the App

```bash
npm run dev
```
